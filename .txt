# Flash-message apko ye allow karta hai ki ek aap ek route ka data dusre route mai dikha sako

 (1) install connect-flash
  npm i connect-flash

(2) make sure you setup express session
  npm i express-session
  then in app.js type : const expressSession = require("express-session");
  then neeche in app.js : app.use(expressSession({
    resave: false,
    saveUninitialized: false,
    secret: "hello world"
  }))

 (3) make sure you put connect-flash in app.use() Function below session
   first upar write : const flash = require("connect-flash");
   then below write : app.use(flash());

 (4) kisi bhi route mai flash message bhejne ke liye, aapko request object par flash method call karna hoga.
   example:
   router.get("/example", (req, res) => {
     req.flash("info", "This is a flash message");
   });

# HOW DO I FIND DOCUMENT WHERE AN ARRAY FIELD CONTAINS ALL SET OF VALUES
